ADV Message Movie Tool

透過メッセージウィンドウ＋テキストアニメーションの連番PNGをZIPで書き出すブラウザアプリです。

今回の更新内容:
- 複数シーン管理に対応しました。
- 左側の素材設定より上に「シーン管理」を追加しました。
- 1プロジェクト内に複数シーンを作成し、プルダウンで切り替えできます。
- シーンの追加、名前変更、複製、削除に対応しました。
- セリフ一覧と音声ファイルはシーンごとに保存されます。
- キャラクター登録、メッセージウィンドウ画像、表示設定はプロジェクト共通です。
- JSON保存・キャッシュ保存時に全シーンの情報を保存します。
- 旧形式JSONを読み込んだ場合は、自動で「シーン1」に変換します。

使い方:
1. index.html をブラウザで開きます。
2. シーン管理でシーンを追加・選択します。
3. 現在のシーン用の音声を読み込みます。
4. キャラクターを登録します。
5. セリフ入力欄から名前・セリフ・開始秒・終了秒・文字送り速度を設定して追加します。
6. 必要に応じて別シーンへ切り替え、別の音声とセリフを登録します。
7. JSON保存またはキャッシュ保存でプロジェクト全体を保存します。
8. 書き出しは現在選択中のシーンが対象です。

補足:
- キャンバスサイズは 1920×1080 です。
- 書き出しPNGは背景透過です。
- プレビュー背景は確認用で、書き出しには入りません。
- 音声をJSONに含めるため、長時間音声を複数シーンに入れるとJSONが大きくなります。
- ZIP書き出しにはJSZip CDNを使用しているため、インターネット接続が必要です。
