ADV Message Movie Tool

ADVノベルゲーム風のメッセージウィンドウ＋テキストアニメーションを、背景透過の連番PNG ZIPとして書き出すブラウザアプリです。

今回の更新内容:
- UI Languageを追加し、日本語 / English / 한국어 のUI切り替えに対応しました。
- 入力テキスト言語を追加し、台詞txt読み込みの案内を日本語 / English / 한국어 に対応しました。
- 台詞txt読み込みで、日本語・英語・韓国語のキャラクター名と本文を扱えるようにしました。
- JSON保存・キャッシュ保存にUI言語と入力テキスト言語を含めるようにしました。
- 音声付きMP4 / M4V / MOVの読み込みに対応しました。
- MP4読み込み時、プレビューキャンバスの下敷きとして映像を表示します。
- テキスト・メッセージウィンドウのプレビューはMP4再生位置と同期します。
- MP4映像は確認用の下敷きで、透過PNG書き出しには含まれません。
- 任意フォント読み込みに対応しました。
- 読み込んだ任意フォントは、名前フォント・本文フォントに別々に設定できます。
- 任意フォントはJSON保存・キャッシュ保存に含まれます。
- 本文アニメーションに「拡大しながら表示」を追加しました。
- 拡大しながら表示は、1文字ずつ小さいサイズから通常サイズへ拡大しながら表示されます。
- 文字送り/秒スライダーで拡大表示の出現速度をリアルタイム調整できます。
- 拡大開始サイズスライダーで、表示開始時の文字サイズを調整できます。

主な機能:
- 1920×1080キャンバス
- メッセージウィンドウ画像読み込み
- 音声 / MP4読み込み
- 音声 / MP4も含めたJSON保存・キャッシュ保存
- 複数シーン管理
- キャラクター登録
- txtから台詞ボタン自動生成
- UI言語切り替え：日本語 / English / 한국어
- 入力テキスト言語：日本語 / English / 한국어
- 台詞ボタンのクリック挿入・再クリックキャンセル
- 透過PNG連番ZIP書き出し
- 名前/本文の位置・サイズ・色・フチ・影・フォント調整
- 本文アニメーション：通常表示 / タイプライター表示 / 拡大しながら表示

任意フォントの使い方:
1. 左側の「フォント登録」を開きます。
2. ttf / otf / woff / woff2 ファイルを読み込みます。
3. 左側の「表示設定」にある「名前フォント」「本文フォント」のプルダウンから、読み込んだ任意フォントを選択します。
4. 名前と本文で別々のフォントを設定できます。
5. JSON保存またはキャッシュ保存すると、読み込んだフォントも一緒に保存されます。

本文アニメーションの使い方:
1. 右側のセリフ入力パネルにある「本文アニメーション」を選びます。
2. 「拡大しながら表示」を選択します。
3. 「文字送り/秒」で文字が出る速度を調整します。
4. 「拡大開始サイズ」で、文字が何%のサイズから拡大して表示されるかを調整します。
5. 入力中・編集中のセリフはプレビューにリアルタイム反映されます。

台詞txtの書き方:
キャラ名：台詞
Character: Line text
캐릭터: 대사
キャラ名「台詞」
キャラ名(台詞)
001 キャラ名：台詞
001 Character: Line text
001 캐릭터: 대사

例:
001 須黒歩生：おはよう
002 Akio Suguro: Good morning
003 스구로 아키오: 안녕하세요
004 須黒歩生「今日も来てくれたんだね」
005 須黒歩生(少しだけ、話していく？)

補足:
- txt読み込み時、未登録のキャラ名は自動でキャラクター登録されます。
- 台詞番号がある場合は、台詞ボタンの上に番号が表示されます。
- 挿入済みの台詞ボタンはグレーアウトし、もう一度クリックすると挿入をキャンセルできます。
- プレビュー背景のミント色は確認用で、書き出しPNGには入りません。
- グリッドやガイドは書き出しPNGには入りません。
- 音声・MP4・フォントをJSONに含めるため、大きいファイルを読み込むとJSONサイズも大きくなります。
- MP4は文字入れ確認用の下敷きです。書き出しPNGは従来通りメッセージウィンドウ・名前・本文のみの透過素材です。
- JSZipはCDNから読み込んでいます。ZIP書き出し時はインターネット接続が必要です。
