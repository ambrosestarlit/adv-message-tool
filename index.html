ADV Message Movie Tool 現行版の使い方

このツールでできること

音声または音声付きMP4に合わせて、ADVノベルゲーム風のメッセージウィンドウ・キャラクター名・セリフ本文を表示し、背景透過の連番PNGをZIPで書き出せます。MP4はプレビュー上で文字入れの下敷きとして表示されますが、書き出しPNGには含まれません。キャンバスサイズは1920×1080固定です。

基本の流れ

  1. 「シーン管理」でシーンを作成または選択します。
  2. 「素材」から音声ファイルまたは音声付きMP4、メッセージウィンドウ画像を読み込みます。
  3. 「キャラクター登録」で名前を追加します。
  4. 右側のプレビュー下にある入力パネルで、名前・セリフ・開始秒・終了秒・文字送り速度・本文アニメーションを設定して「セリフ追加」を押します。
  5. 音声 / MP4プレビューで再生位置を確認しながら、必要に応じてセリフを編集します。MP4の場合は映像がキャンバスの下敷きとして同期表示されます。
  6. 「書き出し」から現在選択中のシーンを透過PNG連番ZIPとして書き出します。

シーン管理

1プロジェクト内に複数シーンを作成できます。シーンごとに音声とセリフ一覧が保存されます。キャラクター、メッセージウィンドウ画像、表示設定、台詞ボタンはプロジェクト共通です。

  • 追加：新しい空のシーンを作成します。
  • 名前変更：現在のシーン名を変更します。
  • 複製：現在のシーンの音声とセリフをコピーして新規シーンを作成します。
  • 削除：現在のシーンを削除します。最低1つのシーンは残ります。

台詞txt読み込みと台詞ボタン

txtを読み込むと、1行ごとのセリフをキャラクター別のボタンとして登録できます。未登録のキャラクター名は自動追加されます。

001 キャラ名：おはよう
キャラ名「今日はいい天気だね」
キャラ名(小声で呼びかける)

対応形式は「キャラ名：」「キャラ名「」「キャラ名(」です。先頭に半角数字3桁＋半角空白がある場合、その番号がボタン上に表示されます。ボタンをクリックすると、その台詞が現在時刻にセリフとして追加され、追加済みのボタンはグレーアウトします。もう一度クリックすると挿入をキャンセルできます。

セリフ入力

  • 名前選択：登録済みキャラクターから選びます。最後に選んだ名前は維持されます。
  • セリフ入力：表示したい本文を入力します。
  • 開始秒・終了秒：セリフを表示する時間です。
  • 本文アニメーション：通常表示、タイプライター表示、拡大しながら表示から選べます。
  • 文字送り/秒：タイプライターと拡大表示の出現速度です。数値入力またはスライダーでリアルタイム確認できます。
  • 拡大開始サイズ：「拡大しながら表示」の開始時の文字サイズです。
  • 現在時刻を開始へ／終了へ：音声またはMP4プレイヤーの現在位置を入力欄へ反映します。

任意フォント

左側の「フォント登録」から ttf / otf / woff / woff2 を読み込めます。読み込んだフォントは「表示設定」の名前フォント・本文フォントに追加され、名前と本文へ別々に割り当てできます。JSON保存・キャッシュ保存にも任意フォントが含まれます。

表示設定

左側の「表示設定」から、メッセージウィンドウ画像の位置・サイズ、名前文字、本文文字、折り返し幅、行間、最大行数、文字色、フチ、影、フォントを調整できます。「フォント登録」から任意フォントを読み込むと、名前フォント・本文フォントで個別に選べます。プレビュー上の薄いミント背景は確認用で、書き出しPNGには入りません。

保存と復元

  • JSON保存：現在のプロジェクト全体をJSONファイルとして保存します。
  • JSON読込：保存済みJSONを読み込みます。
  • キャッシュ保存：ブラウザ内に一時保存します。
  • キャッシュ復元：ブラウザ内の保存データを復元します。

音声・MP4も保存対象です。長い音声やMP4、複数シーン分の素材を含めると、JSONやキャッシュの容量が大きくなります。

書き出し

書き出しは現在選択中のシーンが対象です。FPS、開始秒、終了秒、ファイル名接頭辞を指定し、「透過PNG連番ZIPを書き出し」を押してください。MP4映像は確認用の下敷きなので、PNGには含まれません。

ZIP書き出しにはJSZip CDNを使用しているため、インターネット接続が必要です。
